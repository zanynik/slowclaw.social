Phase 04: Get Started

Quick Start

1. Prerequisites

Required: Rust toolchain (rustc, cargo)

Optional (for rebuilding the web UI): Node.js 18+

2. Build

cargo build --release

Binary path: ./target/release/slowclaw

3. Run

./target/release/slowclaw gateway

Or for full scheduling + chat worker runtime:

./target/release/slowclaw daemon

Native apps are currently in development to bring the full experience to your devices.
